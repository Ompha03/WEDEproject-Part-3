# WEDEproject-Part-3

Made In Venda is a fully responsive brand website showcasing premium streetwear inspired by Venda culture and tradition.
The website features a modern dark theme with a sticky header, mobile-friendly navigation, and a product catalog with pricing.
It includes multiple pages, including Home, About, Articles, Store with Male and Female categories, Checkout, and Contact pages with Google Maps integration. The site is built with HTML, CSS, and JavaScript, and uses Boxicons for icons.

The website is fully responsive across all devices (desktop, tablet, and mobile) with a hamburger menu that appears on smaller screens.
Key features include a search bar, product grid with hover effects, and a contact form.
To run the site locally, simply open Home.html in any modern browser or use VS Code's Live Server extension. No build process or dependencies are required - clone the repository and open the file.
All images are stored in the Images/ folder, and the main stylesheet is style.css, with all JavaScript functionality in main.js.
